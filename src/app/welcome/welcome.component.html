MonodiZero

A standalone note editor for monophonic medieval chant. Runs entirely in your browser — no server, no login required. Built for the creation of scholarly chant editions, with optional GitHub integration for storage and collaborative work.

Zero-Server Architecture

All data — sources, documents, neumes — is stored locally in your browser using IndexedDB. No backend, no account, no dependency on external infrastructure.

GitHub Integration

Connect to a GitHub repository to push and pull your workspace. This enables cloud backup and collaborative editing across multiple researchers.

PDF & MonodiJSON Export

Export editions as typeset PDF with configurable layout settings, or as MonodiJSON (Markdown-compatible) for long-term archival. MEI export is planned for a future release.

OMMR4all Import

Read files produced by the OMMR4all optical music recognition system, allowing you to start from machine-generated transcriptions and refine them manually.

Source & Document Search

Search across sources and documents using full-text filtering. Quickly locate chants by initium, feast day, genre, or folio reference.

Critical Apparatus

Attach structured comments, textual variants, and notes to individual neumes or syllables. Exported into a formatted critical apparatus in the PDF output.
