Comments: old and new format

Here you can enter comments on notes. Each comment consists of up to 4 parts:

  • A text comment
  • An emendation flag
  • A staff line
  • A grid

The text comment, emendation, and staff line form the legacy format. The emendation flag and grid form the new format.

For new comments, please use only the emendation flag and the grid. Legacy comment parts may remain as-is and do not need to be removed. End users on the Corpus Monodicum website will only see the legacy parts if no new-format content exists.

Structure of a new comment

A new comment consists of a series of (possibly nested) elements. The available elements are:

  • A grid, to arrange other elements side by side
  • A text block, where you can enter free text
  • A square bracket
  • A staff line
Example of a new comment

To create a comment showing the original notes on the left, then a bracket, then a corrected staff line, proceed as follows:

  1. Click "Add Grid".
  2. Since you want to arrange 3 sub-elements, click "Grid".
  3. Add three columns to the grid via the "+" button.
  4. In the first column, click "Notes". The original line is inserted automatically.
  5. In the second column, click "Bracket".
  6. In the third column, click "Notes".
  7. You can now edit the right-hand staff line freely.

Done. You can now close the comment dialog and save the document.

Text formatting

In text blocks you can not only enter plain text, but also apply Markdown-like formatting. Text is displayed to end users in italics by default. Additional formatting options:

  • Normal (non-italic) text: wrap in (( and )).
  • Small-caps text: wrap in [[ and ]].
  • Framed text: wrap in {{'{{'}}{{'}}'}} .
